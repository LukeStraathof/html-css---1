Stap 3

Nadat je op enter hebt gedrukt krijg je een lijst met links. Op die links kan je klikken. Hier onder staat een voorbeeld van hoe zo'n lijst er uit ziet. De blauwe zinnen/woorden zijn de links waar je op kan klikken. Klik op een link waarvan je denkt dat dat de goede informatie gaat geven. Nadat je dat hebt gedaan kom je op een website zoals deze en kan je de informatie die je zoekt daarop vinden als je alle stappen goed hebt gedaan.

Als je alweer bent vergeten wat je moet doen heb ik hier nog een tabel voor je.
